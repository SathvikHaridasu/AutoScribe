Documentation

Quick Start Guide

  1. Launch AutoScribe
  2. Enter or paste the text you want to type
  3. Set your desired typing speed (WPM)
  4. Adjust the countdown delay if needed
  5. Click Start or press F6
  6. Switch to your target window during countdown
  7. Watch as the text is typed automatically

Default Hotkeys

  • F6 - Start typing
  • F7 - Stop typing
  • F8 - Pause/Resume typing
